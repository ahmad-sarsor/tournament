تبويبات مقسّمة داخل كبسولة — التنقّل بين أقسام البطولة (البرنامج / الترتيب / الفرق / الإحصائيات).

```jsx
<Tabs
  items={[{ id: "schedule", label: "البرنامج" }, { id: "standings", label: "الترتيب" }]}
  active={tab}
  onChange={setTab}
/>
```

التبويب المفعّل أزرق ممتلئ (وليس أبيض كالكود الأصلي — قرار القشرة المرِحة). الكبسولة تلتف أفقياً عند الضيق.
