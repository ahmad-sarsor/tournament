شارة حالة (حبّة دواء) — حالات البطولة قادمة/جارية/منتهية، «مباشر» النابضة، والذهبية للجوائز.

```jsx
<Badge status="active" />            {/* جارية — أخضر */}
<Badge status="live" />              {/* مباشر — أحمر نابض مع نقطة */}
<Badge status="upcoming" />          {/* قادمة — رمادي */}
<Badge status="gold">🏆 البطل</Badge> {/* ذهبية */}
```

النص الافتراضي يأتي من الحالة (نصوص `i18n.js` الأصلية). مرّر `children` لتخصيصه.
