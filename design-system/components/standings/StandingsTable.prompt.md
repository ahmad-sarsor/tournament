جدول ترتيب بيت واحد — الأعمدة الأصلية من `render.js`: # / الفريق / لعب / ف / ت / خ / له / عليه / الفارق / نقاط / السجل.

```jsx
<StandingsTable
  qualifiers={2}
  showExtra
  rows={[
    { rank: 1, name: "خزيمة", played: 4, won: 3, drawn: 1, lost: 0, gf: 9, ga: 3, points: 10, form: ["w","w","d","w"] },
    { rank: 2, name: "عثمان بن عفان", played: 4, won: 2, drawn: 1, lost: 1, gf: 6, ga: 4, points: 7, form: ["l","w","d","w"] },
  ]}
/>
```

- المركز الأول: خلفية ذهبية خفيفة + شريط ذهبي بمحاذاة البداية.
- المتأهّلون (`qualifiers` الأوائل): خلفية زرقاء خفيفة + شريط أزرق.
- `showExtra` يُظهر له/عليه/الفارق/السجل (مخفية افتراضياً — عرض مضغوط كالموبايل).
- `medals` يستبدل 1/2/3 بـ 🥇🥈🥉 (نمط لوحة المتوقّعين).
- عنوان فوق الجدول: `<div className="standings-title">🛡️ البيت الأول</div>`.
