شجرة خروج المغلوب — أعمدة من اليمين لليسار (RTL): ربع النهائي ← نصف النهائي ← النهائي الذهبي المتوَّج بـ🏆.

```jsx
<Bracket rounds={[
  { title: "نصف النهائي", matches: [
    { home: "خزيمة", away: "عمر بن الخطاب", hs: 2, as: 0, status: "finished" },
    { home: "بلال بن رباح", away: "علي بن أبي طالب", hs: 1, as: 1, status: "live" },
  ]},
  { title: "النهائي", matches: [{ home: "خزيمة", away: null }] },
]} />
```

- الفائز: خلفية زرقاء خفيفة + اسم أسود سمين (وفي النهائي: ذهبية).
- `live`: حلقة حمراء حول البطاقة. فريق غير محسوم: «يُحدَّد لاحقاً».
- أسماء الجولات من الكود: النهائي، نصف النهائي، ربع النهائي، دور الـ16.
- تلتف أفقياً بتمرير (`bracket-scroll`) على الشاشات الضيقة.
