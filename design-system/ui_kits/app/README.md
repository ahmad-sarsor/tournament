# UI Kit — التطبيق العام (منصّة البطولات)

إعادة بناء تفاعلية للواجهة العامة من https://github.com/ahmad-sarsor/tournament بالقشرة المرِحة الجديدة، بنفس البِنى والحقول من `render.js` / `app.js`.

## التدفّق

1. **قائمة البطولات** (`TournamentsScreen.jsx`) — شبكة بطاقات `TournamentCard` مع شارة حالة وشريط تقدّم.
2. **البطولة** — تبويبات `البرنامج | الترتيب`:
   - **البرنامج** (`ScheduleScreen.jsx`) — فلتر بيوت + مباريات مجمّعة بالأيام (`day-head` + `MatchRow`)، منها مباراة **مباشرة**.
   - **الترتيب** (`StandingsScreen.jsx`) — `StandingsTable` لكل بيت + مفتاح الألوان.
3. **المباراة** (`MatchScreen.jsx`) — `Scoreboard` كبيرة + الأحداث على خطّ منتصف (`mp-events`: ⚽ 🟨 🟥) — انقر المباراة المباشرة لرؤيتها.

`data.js` — بيانات «أحفاد الصحابة» الحقيقية من `seed-data.js` (أسماء الفرق والبيوت والجدول).

`mobile.html` — نفس التطبيق بمعاينة هاتف 390px (الاستخدام الأساسي للمنتج) — جدول مضغوط + زر «تفاصيل أكثر»، فلاتر بعرض الشاشة، أهداف لمس 44px.
